# Iris Flower Classification using MLP from Scratch

## Project Overview

This project implements a Multi Layer Perceptron (MLP) Neural Network from scratch using NumPy without TensorFlow or PyTorch.

The model is trained on the Iris dataset and deployed using Streamlit.

---

## Features

- Forward Propagation
- ReLU Activation
- Softmax Output Layer
- Backpropagation
- Cross Entropy Loss
- Training Visualization
- Streamlit Deployment

---

## Dataset

Iris Dataset from Scikit-Learn

Features:
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

Classes:
- Setosa
- Versicolor
- Virginica
